Privacy Policy for MusicBot

1. Introduction
This Privacy Policy explains how MusicBot collects, uses, and protects information when you use the MusicBot Discord application. By using MusicBot, you agree to the practices described in this policy.

2. Information We Collect
MusicBot collects and processes the following types of information:

User Commands: We collect commands entered by users to execute bot features such as playing music, managing playlists, or retrieving information. This data is processed in real-time and not stored unless specified.

Server Information: Basic information about the server where MusicBot is added, such as server ID and channel IDs, may be temporarily stored for functionality.

Favorites and Settings: When you save a favorite song or configure MusicBot’s settings, this data is stored securely to provide persistent features.

3. How We Use Information
The information collected by MusicBot is used solely for the purpose of providing and improving bot functionality:

Executing user commands and responding in Discord.

Saving user preferences (e.g., favorite songs) and bot settings.

Debugging and monitoring bot performance to ensure stability.

4. Data Retention

Temporary data, such as user commands and server information, is deleted after processing.

Persistent data, such as saved favorites and configuration settings, is stored until explicitly removed by the user or server administrator.

5. Data Sharing and Security

No Third-Party Sharing: We do not share collected data with third parties.

Security Measures: We implement industry-standard security practices to protect stored data. However, no system is entirely secure, and we cannot guarantee absolute security.

6. User Rights

You have the right to request deletion of stored data related to your usage of MusicBot. Contact our support team to make such a request.

Server administrators can manage and delete server-specific data, including user preferences.

7. Cookies and Tracking
MusicBot does not use cookies or tracking technologies. All interactions occur within Discord’s ecosystem.

8. Changes to This Privacy Policy
We reserve the right to update this Privacy Policy at any time. Changes will be effective immediately upon posting. Continued use of MusicBot signifies acceptance of the updated policy.

9. Contact Information
If you have questions or concerns about this Privacy Policy, please contact the MusicBot support team through the designated Discord server or other specified communication channels.
